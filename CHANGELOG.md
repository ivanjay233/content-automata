# Changelog

## [0.1.0] - 2026-06-02

### Added
- Initial release of content-automata
- Topic research stage with Tavily and Exa provider support
- Copywriting stage with blog, social, and ad copy variants
- Image generation stage with OpenAI DALL-E and Stability AI support
- Scheduling stage with Markdown, HTML, and CSV export
- CLI interface with `init`, `run`, `schedule`, and `status` commands
- Pydantic models for content pipeline data structures
- Configurable pipeline via YAML configuration
- Simulated fallback modes for development without API keys
- Basic test suite for all pipeline stages
- GitHub Actions CI with linting and testing
- Makefile for common development tasks
